# computerGraphics

A public repository for computer graphics experiments, assignments, and demos.

## What's inside
- Rasterization and ray-tracing exercises
- Shaders and rendering pipeline experiments
- Small demos and notes

## Getting started
Clone the repository and explore the subfolders. Many demos/projects contain their own instructions.

```bash
git clone https://github.com/your-username/computerGraphics.git
cd computerGraphics
```

If a project has a `README` or build script inside its folder, follow those steps there.

## Contributing
Issues and pull requests are welcome. If you spot a bug or want to propose an improvement, please open an issue describing the change first.

## Notes
- This repository may include CPU and GPU based experiments; some samples might require a modern GPU and recent graphics drivers.
- Developed on Windows with WSL/PowerShell; behavior may vary across platforms.
